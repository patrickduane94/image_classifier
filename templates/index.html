This image classifier is a convolutional neural network developed with the PyTorch framework for deep learning. The dataset used to train and test was downloaded from Kaggle. It contains around 16,000 images of faces of wild and domesticated cat and dog species. Before training, the images were resized to 128x128 pixels to preserve detail while also keeping computational cost relatively low.

There are three convolutional layers which use 3x3 filters, each doubling the number of features learned in the previous layer (32->64->128). Training went on for 10 epochs using cross-entropy loss and the Adam optimizer, which combines momentum and adaptive learning rates to improve training efficiency during backpropagation.
